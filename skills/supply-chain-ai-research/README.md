# supply-chain-ai-research

供应链 AI 行业深度研究框架。基于横纵分析法（Horizontal-Vertical Analysis），针对供应链+AI交叉领域定制。

支持：
- 多源数据交叉验证（aihot + Brave Search + arXiv + RSS）
- 纵轴：时间线演变分析
- 横轴：同期竞品对比
- 横纵交汇：独到洞察产出
- 周报自动生成（5板块 x 3条 = 15条 + 洞察）

## 文件说明

```
supply-chain-ai-research/
├── SKILL.md                  # 主技能文档
└── references/
    └── data-sources.md       # 数据源策略（搜索关键词、RSS源、去重规则）
```

## 安装

```bash
cp -r supply-chain-ai-research ~/.hermes/skills/domain/supply-chain-ai-research
```

## 适用场景

- 供应链/物流行业的 AI 应用跟踪
- 竞品分析（仓储自动化、无人驾驶货运、物流SaaS等）
- 行业周报自动生成
- 供应链+AI 的深度研究报告
